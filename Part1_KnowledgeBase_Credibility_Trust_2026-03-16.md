## 4. Trust Signals: Data Security & Compliance

### Data Ownership & Privacy
- All data belongs to the customer, not to GeoDin or Fugro. GeoDin does not host, access, anonymize, reuse, or sell customer data.
- Customer-controlled data residency: data can be stored on-premises (own network), on the customer's cloud infrastructure, or on a local device.
- GeoDin does not access customer data. Data put into GeoDin is not visible to Fugro or other clients.
- The platform can run completely offline on a laptop with a local database, suitable for defence ministries and organizations requiring data behind strict security layers.
- NDA arrangements are available for clients who need to share sensitive project data during configuration.

### Database Security & Infrastructure
- Supports enterprise-grade database backends: Oracle, Microsoft SQL Server, PostgreSQL, MySQL.
- Client-server SQL databases have no practical size restriction (hundreds of gigabytes).
- Multiple concurrent users can collaborate on the same database simultaneously.
- Two deployment models: on-premises (most common) or cloud-hosted on customer's own infrastructure.
- GeoDin provides setup guidance but does not host data by default -- the customer retains full control.

### Standards Compliance
- Automatic and enforced adherence to 11 international geotechnical standards: DIN 4022/23, DIN EN ISO 14688/89, DIN 4943, SEP1, SEP3, DIN EN ISO 22475, NEN, ONORM, GOST, KA5, ASTM, BS 5930.
- AGS 4 compliance: supports both AGS 4.0.4 and AGS 4.1.1 formats with official recognition from the AGS Committee.
- 28 AGS location data types and 58 AGS sample data types.
- Data traceability maintained from field collection through to final deliverable, with standard compliance enforced at every step.

### European Data Residency
- Positioned as offering "European-grade data residency and compliance."
- Customer chooses where data is stored -- can keep data entirely within their own country or region.
- Key differentiator vs. cloud-only competitors: no mandatory cloud hosting, no data leaving the customer's control.
